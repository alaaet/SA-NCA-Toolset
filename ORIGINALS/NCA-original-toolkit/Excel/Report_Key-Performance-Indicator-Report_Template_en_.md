## Instructions
| <ORGANIZATION NAME> - KEY PERFORMANCE REPORTING TEMPLATE |
| --- |
| INTRODUCTION |
| PURPOSE: This tool is designed to show performance across Key Performance Indicators (KPIs) selected by <organization name> in relation to targets or goals for these indicators. The document is intended to be updated with actual data from the <organization name>'s systems of record on a quarterly basis after the data has been reviewed, reconciled and/or closed, as applicable. The user defines both the KPIs and the targets against which they are measured. |
| DATA ENTRY INSTRUCTIONS ("KPI" tab): |
| Column "D" (Cybersecurity Domain) - The section is determined by <organization name>'s cybersecurity domains, in order to categorise the key performance indicators being used by <organization name>\n |
| Column "E" (Key Performance Indicator (KPI)) - The section collects all the topics of key performance indicators, that are being measured at <organization name>\n |
| Column "G" (KPI Definition) - The section defines all the exact key performance indicators, that are being measured at <organization name>\n |
| Column "H" (KPI Type) - The type of the indicator, categorized by the method of measurement (e.g., Count, Percentage, Sum, Average Ratio)\n |
| Column "I" (Frequency) - The time interval of measurement decided or defined of the indicator (e.g. Daily, Weekly, By-weekly, Monthly, Quarterly, Annually)\n |
| Column "J" (Data source) - This section serves to determine the source of the data being collected (e.g., industrial or enterprise historians, relational databases, external web services, manually entered data, business applications, devices)\n |
| NaN |
| MEASUREMENT INSTRUCTIONS ("Measurement table" tab): |
| Column "C" (kpi\_id) - The column shall be used to move KPI metrics from the KPI sheet. Only the number of the KPI shall be written into the table, the related information is automatically filled for each record in the "D" (Cybersecurity Domain), "E" (KPI Definition) and "F" (KPI Type) columns.\n |
| From column "G" to "S", the values shall be administered manually. |
